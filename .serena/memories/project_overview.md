Personal website source for https://bogorad.github.io built with Hugo. Main editable content lives in `content/`, `data/`, `layouts/`, `assets/`, and `static/`. Key data files: `content/_index.md` for homepage copy and profile details, `data/contacts.toml` for contact links/icons, `data/timeline.toml` for timeline items. Templates live under `layouts/`, especially `layouts/_partials/home/`. Styling is in `assets/css/style.css`. Hugo config is in `hugo.toml`.
